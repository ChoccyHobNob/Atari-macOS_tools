Description:
------------
Loads Basic programs from H1:
Program based on SidLoad. Does the same thing for Basic programs.


Usage:
------
BasLoad 

Longer version asks for filename on H1: drive.
Shorter version loads H1:PRG.BAS file. I'm using it for Atari800 emulator
for automatic game loading from Perl scripts.
